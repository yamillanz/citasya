# Initialize OpenSpec for Claude + Cursor

```bash
cd your-project
openspec init --tools claude,cursor
```

Creates:
- `openspec/` (specs, changes, config)
- `.claude/skills/` (Claude Code skill files)
- `.cursor/rules/` (Cursor rules)

Use **openspec-onboard** for a guided walkthrough, or **openspec-new** to start your first change.
